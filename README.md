## اسکریپت ساخت کانفیگ وایرگارد برای v2rayng و MahsaNG 
---
## قابلیت ها

### ساخت کانفیگ وایرگارد وارپ
### اسکن خودکار IPv4 و IPv6
### قابلیت استفاده از آیپی و پورت مشخص شده توسط کاربر
### قابلیت تغییر mtu و نام کانفیگ توسط کاربر
